About the project

This project was created as part of a school assignment to create a clone of the game Wordle. It includes three main pages: one for playing the game, another for displaying highscores, and a third for project details.

The aim was to learn how to build a full-stack application. The game's interface was made using React, while the game's logic takes place on the server using Express.

Highscores are stored in a MongoDB database using Mongoose, and the highscore page is rendered on the server using EJS.
